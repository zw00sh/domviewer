Spider Payload Deep Dive

Feb 3, 2026

The spider payload crawls same-origin links starting from the current page. It discovers the site structure and reports URLs back to the server.

Crawl Strategy

The spider uses a breadth-first approach:

  1. Extract all <a href> links from the current document
  2. Filter to same-origin URLs only
  3. Queue undiscovered URLs for fetching
  4. Report each discovered URL with its depth level

Message Format

{
  "type": "result",
  "url": "https://example.com/page",
  "depth": 1,
  "links": ["https://example.com/other"]
}

The spider respects depth limits and avoids re-crawling already-visited URLs. Results are streamed to all connected viewers in real time.

Try it on the home page to see it discover all the pages in this test site, including the product and about sections.
