Getting Started with domviewer

Jan 15, 2026

domviewer captures the DOM state of a target page and streams it in real time using Yjs for efficient delta synchronization.

How It Works

The loader script iframes the target page and establishes a WebSocket connection to the server. Payload modules are sent over the wire and dynamically loaded on the client.

Key Concepts

  • Iframe isolation keeps the loader separate from target page scripts
  • MutationObserver tracks all DOM changes
  • Yjs CRDTs enable efficient binary delta sync
  • Pluggable payloads allow modular functionality
